Informes y Pagos de Consultores

Información del Sistema

Este sistema optimiza la gestión digital de documentos y solicitudes de pago de los consultores, permitiendo un control eficiente y ágil de contratos y formularios, garantizando transparencia en el proceso.

El frontend fue desarrollado con Angular 15 y TypeScript, utilizando PrimeNG para componentes avanzados y Bootstrap 5 para un diseño responsivo, implementando Route Guards para controlar el acceso según roles de usuario.

En el backend, se utilizó Laravel 11 con ORM Eloquent para la gestión eficiente de datos, y PostgreSQL como base de datos.

La autenticación segura se implementó con JSON Web Tokens (JWT).

El control de versiones se gestionó con Git y GitHub, y la comunicación entre el frontend y el backend se realizó mediante API REST.

Se utilizó Docker para la creación y gestión de contenedores, mejorando el despliegue y la escalabilidad del sistema.

Módulos desarrollados en el sistema:

  • Módulo de Requisitos
  • Módulo de Usuarios
  • Módulo de Proyectos
  • Módulo de Dependencias
  • Módulo de Consultores
  • Módulo de Contratos
  • Módulo de Solicitudes de Pago
  • Módulo de Documentos del Contrato
  • Generación de PDF de Documentos

Parqueo y Tarifas del SEM

Información del Sistema

Este sistema permite a los funcionarios del SEM (Servicio de Estacionamiento Municipal) gestionar el parqueo y tarifas, facilitando el pago digital a través de móviles o parquímetros, y optimizando el control de las zonas de parqueo.

El frontend se desarrolló inicialmente con Angular 14 y TypeScript, pero fue actualizado a Angular 16 para aprovechar las últimas características. Se emplearon PrimeNG para componentes avanzados y Bootstrap 5 para garantizar un diseño responsivo, además de implementar Route Guards para controlar el acceso según roles de usuario.

El backend se implementó con Node.js y Express 4, empleando Express Validator para la validación de datos y Moment.js para el manejo de fechas. La base de datos se gestionó con MySQL y Sequelize 6 como ORM, optimizando la gestión de entidades y relaciones.

Se utilizó JSON Web Tokens (JWT) para garantizar una autenticación segura y proteger el sistema.

Se implementó Puppeteer para generar documentos PDF personalizados desde contenido HTML.

El control de versiones se gestionó mediante Git y GitHub, y se implementó API REST para la comunicación entre el frontend y el backend.

Se utilizó Docker para la creación de contenedores, mejorando el despliegue y la escalabilidad del sistema.

Módulos desarrollados en el sistema:
  • Módulo de Personal
  • Módulo de Zonas
  • Módulo de Calles
  • Módulo de Restricciones
  • Módulo de Reportes
  • Módulo de Usuarios
  • Servicios API REST

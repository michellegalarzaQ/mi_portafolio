Inspección de Gases Vehiculares

Información del Sistema

Este sistema optimiza la gestión de inspecciones de gases vehiculares, permitiendo solicitudes en línea o presenciales, pagos ágiles mediante códigos QR y consulta de placas a través del RUAT.

El frontend fue desarrollado con Angular 16 y TypeScript, utilizando PrimeNG para componentes avanzados y Bootstrap 5 para garantizar un diseño responsivo. Se implementaron Route Guards para proteger las rutas y controlar el acceso según roles de usuario.

En el backend, se utilizó inicialmente Laravel 9 con ORM Eloquent para una gestión eficiente de los datos, pero posteriormente se actualizó a Laravel 10 para aprovechar sus últimas características. El sistema de base de datos utilizado fue PostgreSQL.

La autenticación segura se garantiza mediante JSON Web Tokens (JWT).

El control de versiones se gestionó con Git y GitHub, y la comunicación entre el frontend y el backend se realizó mediante API REST.

Se utilizó Docker para la creación y gestión de contenedores, mejorando el despliegue y la escalabilidad del sistema.

Módulos desarrollados en el sistema:
  • Módulo de Usuarios
  • Módulo de Personal
  • Módulo de Centros de Inspección
  • Módulo de Líneas
  • Módulo de Horarios de las Líneas
  • Módulo de Solicitudes de inspección
  • Módulo de Mediciones (Ruido y Emisión por Combustible)
  • Módulo de Reportes
  • Gráficos y Generación de PDF
  • Servicios API REST
  • Consumo de Servicios RUAT
  • Consumo de Servicios Pagos QR

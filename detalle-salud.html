Citas y Referencias Médicas

Información del Sistema

Desarrollé este sistema utilizando tecnologías modernas para optimizar la gestión de consultas médicas y referencias en centros de salud.

El frontend se desarrolló con Angular 16 y TypeScript, empleando PrimeNG 16 para componentes avanzados y Bootstrap 5 para garantizar un diseño responsivo, además de implementar Route Guards para controlar el acceso según roles de usuario.

El backend fue desarrollado con Node.js 18 y Express, utilizando Express Validator para validar datos y Moment.js para gestionar fechas. La base de datos se gestionó con PostgreSQL y Sequelize 6 como ORM, optimizando la gestión de entidades y relaciones.

Para la creación de reportes dinámicos, utilicé PDFMake para generar documentos en formato PDF.

Implementé JSON Web Tokens (JWT) para asegurar una autenticación confiable y garantizar la seguridad del sistema.

El control de versiones se realizó con Git y GitHub, y se empleó API REST para la comunicación entre el frontend y el backend.

Se utilizó Docker para la creación y gestión de contenedores, mejorando el despliegue y la escalabilidad del sistema.

Módulos desarrollados en el sistema:
  • Módulo de Referencias Médicas
  • Módulo de Consultas Médicas
  • Módulo de Médicos
  • Módulo de Especialidades
  • Módulo de Horarios
  • Módulo de Consultorios
  • Módulo de Reportes de Referencias
  • Módulo de Reporte de Consultas
  • Módulo de Reporte de Fichas
  • Gráficos y Generación de PDF
  • Servicios API REST para la Aplicación Móvil
